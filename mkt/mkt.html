MIKROTIK

TUTORIAL MIKROTIK DENGAN VirtualBox



Konfigurasi yang di pelajari:
1. Install Mikrotik
2. Konfigurasi Dasar Mikrotik
3. Block Extension (.mp3|.mp4)
4. Block Situs (http & https)
5. Timer (Schedule)

Pertama buka VirtualBox dan buat Virtual Machine dan beri nama yang mengandung kata
mikrotik. Setelah kalian buat Virtual Machine kalian ke setting lalu insert file Iso Mikrotik ke
dalam Storage pada settings.

Setelah itu kalian buka Network dan kalian ubah dari NAT ke Bridged Adapter pada Adaptor 1.
Dan pada Adaptor 2 kalian ubah NAT > Internal Network. (opsional jika kalian ingin mengganti
nama adapter)


Selanjutnya Kalian Start untuk menginstall mikrotik tersebut. Jika sudah sampai bagian Router
Software Instalation kalian klik huruf ”a” pada keyboard kalian agar semua settingan pada
mikrotik dapat terinstall


Setelah itu klik huruf ”i > n > y”


Setelah terinstall, sebelum kalian klik Enter, kalian lepas iso mikrotik agar kalian tidak
mengulang penginstallanya kembali


Login terlebih dahulu dengan user ”Admin” dengan password (kosong)


Jika sudah login, saat see software license kalian klik n > enter


Setelah itu kalian buka winbox pada laptopkalian dan login menggunakan Mac Address karena
dhcp client mikrotik kalian belum terhubung jadi belum bisa login menggunakan IP


Setelah connect ke dalam mikrotik, hal pertama yang harus kalian lakukan adalah konfigurasi
dasar mikrotik agar mendapat internet. Tambahkan DHCP Client pada menu IP > dhcp client


Setelah itu kalian klik ”+” untuk menambahkan dhcp client dan klik Apply > ok. Setelah
menambah dhcp client kita skrng mendapatkan ip dari wifi laptop kita.


Dengan begini kita sudah bisa masuk mikrotik dengan menggunakan IP


Lalu setelah menambahkan dhcp client, kita akan membuat IP pada Addresses untuk port adapter
ke 2 pada mikrotik pada menu ip > addresses. Dan klik ”+” unutk menambah Address. Isi
address sesuai keinginan kalian dan jangan lupa untuk menambahkan (/24). Disini saya
menggunakan ip 10.10.10.1/24 dengan interface ether2 lalu klik Apply > OK


Lalu setelah dari addresses kita akan menambahkan server pada setiap adapter di mikrotik pada
menu ip > dhcp server. Pada dhcp server kita menambahkan server dengan ”DHCP Setup” dan
hanya klik ” next” hingga selesai. Lakukan kembali hingga kedua adapter mendapat dhcp server


Selanjutnya kita akan menambahkan NAT pada menu ip > firewall > NAT


Klik ”+”, pada General isi Chain dengan ”srcnat”, dan Out. Interface dengan ether1(sumber
internet). Pada menu Action ubah ke ”masquerade”, Apply > OK


Lalu setelah dari NAT kita akan menambahkan server DNS pada menu ip > dns. Tambahkan
8.8.8.8 pada server dan aktifkan Allow Remote Request, Apply > OK


Selanjutnya untuk mendapatkan internet kita akan menambahkan gateaway pada Routes pada
menu ip > Routes. Tambahkan ip gateaway sesuai dengan ip ether1, Apply > OK


Setelah konfigurasi di atas seharusnya ether2 kita sudah mendapatkan internet. Sebelum kita
mengaktifkan windows pada virtualbox, kita hubungkan terlebih dahulu adapter2/ether2 pada
mikrotik ke dalam windows tersebut. Buka Setting > Network windows pada virtualbox, lalu
ubah NAT > menjadi “Internal Network”. Ganti “intnet” ke nama adapter2 pada mikrotik kalian
agar windows kalian terhubung dengan mikrotik. Pada Advenced ubah deny > Allow all


Lalu kalian bisa start windows pada virtual box Coba buka menu network and internet dan coba
cek network2 apakah sudah mendapatkan ip dari adapter2 pada mikrotik. Usahakan IpV4 kalian
Otomatic, jangan Static IP


Sekarang mari kita coba ping pada cmd di windows. Masuk ke menu run dengan klik
Windows+R, ketik cmd dan enter. Lalu coba ping dns server (8.8.8.8) dan ip (10.10.10.1). Jika
berhasil maka windows kalian sudah mendapat internet


Untuk membuktikan apakah windows sudah benar benar mendapatkan internet kita coba buka
Internet Explorer dan coba search sesuatu. wkwkwk mesti KAYU :)




Selanjutnya kita akan coba untuk memblokir beberapa situs http dan beberapa extensi file seperti
mp3 dan mp4.

Pertama kita buka ”Web Proxy” pada menu ip > web proxy. Aktifkan centang pada ”enabled”
dan ”cache on disk”. Untuk cache administrator bebas kalian bisa tulis terserah kalian. Jika sudah
jangan lupa untuk ”Apply”. Sebelum kita ”OK” kita akan masuk ke dalam menu ”Access”


Di dalam menu Access kita akan menambah situs dan extensi yang akan kita block. Klik ”+”,
lalu isi ”Dst. Port” dengan ”80” dan ”Dst. Host” isi sesuai dengan situs yang akan kalian block,
ubah juga action menjadi Deny. Jika sudah Apply > OK


Untuk memblockir suatu extensi sama halnya dengan block situs, bedanya hanya terletak pada
tempat untuk menuliskannya. Jika pada Dst. Host di gunakan untuk memblock suatu situs, maka
untuk suatu extensi kita akan menuliskannya pada ”Path” dengan catatan diberi tanda bintang (*)
sebelum menuliskan extensi yang akan di block


Selanjutnya agar block situs dan extensi berhasil kita perlu menambahkan NAT baru untuk port
pada Web Proxy. Buka ip > firewall > NAT > ”+”. Ubah chain ke ”dstnat” dan tambahkan
protocol 6 (tcp), lalu isi Dst. Port dengan 80,8080,8000. Jika sudah masuk ke menu action lalu
ubah menjadi ”redirect” dan tambahkan tujuan port ke port web proxy 8080


Sekarang kita akan coba hasil dari block extensi dan situs. Pertama buka windows dan buka
internet explorer. Untuk extensi mp3 kalian bisa search ”mp3 of index” lalu cari web yang
menyediakan file .mp3 sama halnya dgn .mp4.


Jika berhasil maka akan seperti ini.

Situs


Block situs di atas hanya untuk situs http, sedangkan jika ingin block situs https kita akan
menggunakan Layer 7 Protocol. Buka IP > firewall > layer 7 protocol > +


Beri nama terserah kalian lalu isi Regexp dengan

^.+(youtube.com)*.$

Jika ada web lain kalian tinggal menambahkannya saja ke dalam kolom tersebut. Tekan Apply > OK


Selanjutnya buka menu Filter rules pada menu Firewall. Dan tekan ”+” kemudian isi Sec.
Address dengan ip dari ether yang akan kalian block. Lalu buka menu advenced dan isi bagian
layer 7 protocol sesuai dengan nama yang kalian beri pada layer 7 protocol. Setelah itu pergi ke
action dan ubah menjadi ” Drop”. Tekan Apply > OK


Sekarang mari kita cek hasilnya.

Kesimpulan
Block situs dengan Web Proxy dan dengan Layer 7 Protocol, hasilnya berbeda. Perbedaannya
jika pada Layer 7 Protocol situs yang terblock akan terlihat sepeti tidak mendapat internet sama
sekali, sedangkan jika menggunakan Web Proxy situs yang ter block akan langsung terlihat

Selanjutnya kita akan mencoba konfigurasi Timer menggunakan Schedule.

Buka menu Schedule pada System > Schedule


Sebelum lanjut pastikan date and time sesuai. Tekan + lalu beri nama Disable Ether 2 lalu ubah
start time terserah kalian/beberapa menit kedepan. Kalian juga bisa menambahkan interval. Pada
kolom on event isi dengan ”interface disable ether2” perintah tersebut membuat ether 2 kalian
menjadi nonaktif dan tidak bisa memberikan ip. Jadi otomatis windows kalian tidak mendapat internet.


Tambahkan juga Timer untuk meng aktifkan ether2. Sama dengan yang sebelumnya, hanya ganti
pada start timer menjadi beberapa menit kedepan dari yang sebelumnya dan pada on event
menjadi ”interface enable ether2” agar saat tepat pada jam nya, ether2 yang sebelumnya non
aktif menjadi aktif kembali dan windows kembali mendapat internet
